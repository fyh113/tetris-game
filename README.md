# 俄罗斯方块游戏

一个基于 HTML5、CSS3 和 jQuery 开发的经典俄罗斯方块游戏。

## 功能特点

- 🎮 经典俄罗斯方块玩法
- 🏆 历史最高分记录
- 👀 下一个方块预览
- ⌨️ 键盘控制支持
- 💾 本地存储最高分

## 操作说明

- **← →** : 左右移动方块
- **↑** : 旋转方块
- **↓** : 加速下落
- **空格** : 直接落到底部
